Assignment 2 - Raytracer Project (Home)

Features:

  • Render arbitrarily oriented ellipsoids.
  • Render polygons.
  • Use simple Phong Shading (in color).
  • Compute shadows.
  • Compute reflections.
  • Apply linear transformations to objects.
  • Use point and directional lights.
  • Write output to a png file.
  • Axis-Aligned Bounding Boxes for accelerating ray tests.
  • Spot lights.
